안녕하세요! 마케터에서 개발자로 새로운 여정을 준비하고 있는 **황용석**이라고 합니다. <br>
마케터로서 일을 시작했을 때 가슴에 새겼던 **세상에 필요한 가치를 만들고 좋은 영향력을 끼치고 싶다**는 목표와 초심을 잊지 않고 <br>
